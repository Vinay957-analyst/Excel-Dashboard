# Excel-Dashboard

This project presents an interactive Excel dashboard developed using transactional financial data to monitor and analyze business performance. The dataset includes key metrics such as revenue, expenses, profit, region, department, product line, customer segment, and payment method, enabling multi-dimensional analysis.

Using pivot tables, charts, and slicers, the dashboard provides clear insights into revenue trends, regional performance, departmental contributions, and the impact of discounts on profitability. This project demonstrates strong skills in data cleaning, data modeling, and dashboard design to support data-driven decision making
